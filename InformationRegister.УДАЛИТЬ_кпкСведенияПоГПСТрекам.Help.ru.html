Регистр сведений «Сведения по GPS-трекам»

Данный регистр предназначен для хранения информации по GPS-трекам перемещения агентов, полученных из МУ. 

При обмене данными между МУ и ЦБД в регистр заносятся GPS-координаты точек трека, которые фиксируются в МУ посредством функции трекинга. Дополнительно имеется возможность записи скорости перемещения торгового агента, если в МУ включена соответствующая настройка.

Данные в регистр могут заносится в двух режимах:

1. Для каждой точки трека создается отдельная запись с координатами.

2. Загружаются файлы треков из МУ, внутри котрого содержатся записи с координатами треков - данный вариант может быть использован только, если на форме настроек констант "Настройка параметров работы с мобильным приложением" в константе "Приложение просмотра карт" указано использование приложения Map9 Viewer).

На основании данных из этого регистра строятся треки перемещений агентов при формировании отчетов "Карта маршрута агента" и "Контроль передвижений агентов".
