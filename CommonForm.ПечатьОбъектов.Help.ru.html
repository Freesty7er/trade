Форма предназначена для вывода печатных форм. В форме возможен вывод для нескольких объектов сразу. При этом станет доступен список Объекты, с помощью которого можно выполнять навигацию между сформированными печатными формами объектов. Если сформирована только одна печатная форма, список объектов скрывается.

Перед формированием печатных форм производятся различные проверки объектов на корректность данных. Так, документы проверяются на отсутствие пометки удаления, проведённость и т.д. Если какие-то объекты не пройдут проверку, информация о них будет отражена в списке Пропущенные объекты. В случае отсутствия ошибок список скрывается.

При нажатии кнопки Печать макет будет послан на принтер. При печати табличного макета система вначале откроет стандартное окно печати, для текстовых макетов печать будет произведена сразу на принтер. Перед печатью можно выбрать количество экземпляров.
