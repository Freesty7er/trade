Обработка "Генерация файла настроек Агент Плюс СОД"

Данная обработка генерирует XML- файл настроек для службы "Агент Плюс СОД".

Данные для настроек берутся из констант "Папка обмена" и "Адрес" (задаются в форме констант "Настройка параметров работы с мобильным приложением" и регистре сведений «Сведения агентов».

В результате выполнения этой обработки в сгенерированный XML- файл записываются настройки, необходимые для корректного обмена между службой "Агент Плюс СОД", информационной базы 1С:Предприятие и приложением на мобильном устройстве.
