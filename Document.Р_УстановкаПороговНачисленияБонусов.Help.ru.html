Документ "Установка порогов начисления бонусов"

Документ используется для установки шкалы порогов при работе с бонусной системой.

  • Бонусы накапливаются по информационной (бонусной) карте в соответствии со шкалой (пороги), установленной в документе в реквизитах «Нижняя граница (>=)» и «Значение порога».  Начисление бонусов производится по каждой строке чека (только на номенклатуру без флага ""Не применять для бонусов") с округлением до двух знаков после запятой. Начисленные бонусы сохраняются в документе "Продажи по бонусным картам" (создается автоматически при загрузке чека от POS) и в регистре накопления "Продажи по бонусным картам".
  • При необходимости изменить шкалу (полностью или частично) должен быть создан новый документ, в котором должна быть описана ВСЯ новая шкала со всеми порогами.
